    STM32的CAN外设有什么不同？
    其次，STM32与CAN总线之间是采用CAN总线收发器来连接的，它的
    作用是利用两根管脚差分，把单片机自身对外发出的数字信号(0和1)，变
    成两根线(CAN-L和CAN-H)上的3.3V、1.5V、OV，同时也能把CAN总线线上
    的3.3V、1.5V、OV，变成单片机引脚上能识别的数字信号(0和1)。

    因此，32只需要两个引脚事先(RXT和TXD)就可以发送并接收CAN
    TXD：发送0和1，经过收发器十分叉分为差分CAN总线两根线
    RXD：CAN总线两根线经过收发器重新生成0和1到单片机RXD接收
